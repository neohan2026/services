# 跨境电商数据服务

为跨境卖家提供竞品价格监控、评论数据采集、产品调研报告等数据服务。

---

## 可做项目

### 亚马逊 / Shopify 数据采集
- 指定 ASIN / 商品链接 → 返回结构化数据（售价、排名、评论数等）
- 关键词搜索结果批量抓取
- 评论数据导出（含星级、日期、内容）

### 定时价格监控 + 报警
- 设置关注商品列表
- 每日自动检查价格变动
- 降价 / 缺货 / 排名变化时通知

### 产品销量估算
- 基于 BSR（Best Sellers Rank）+ 类目系数估算月销量
- 趋势分析：7 天 / 30 天 / 90 天变化

### 自动化工作流搭建
- 竞品上新自动通知
- 库存预警
- 跟卖监控
